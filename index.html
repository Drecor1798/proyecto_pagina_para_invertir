El dinero disponible es

La idea de este proyecto es crear un simulador de una billetera virtual y dar opcion de inversion que muestre el dinero disponible, las acciones de empresas, criptomonedas y el estado económico de los países con gráficos. Además, debe mostrar el valor de las monedas internacionales tambien ingresar un boton para cambiar la moneda actual, que está en dólares.

También esta pagina tiene el objetivo de mostrar los gastos para los que se uso y tambien diferenciar los ingresos altos y bajos para mostrar las opcciones de inversion
